# 底部反弹
*捕捉形成V形反转并发出复苏信号的股票*

## 1. 什么是此策略？
- **风险等级**: 高
- **Timeframe**: (5-30)
- **Target**: 价值投资者, 逆向交易者

“底部反弹”策略可识别较 52 周高点下跌超过 50% 但今日收盘为正的股票。这是经典的 V 型反转形态——复苏可能已从底部开始。

## 2. 自动筛选参数
当您点击此卡片时，将自动应用以下筛选器：

| 列 | 应用值 |
| :--- | :--- |
| 实时成交量 | > 1.000.000 |
| 市值 | > 10B |

## 3. 工作原理？
将“接近峰值”筛选为最高 50%（远离峰值），将“涨跌幅（%）”筛选为最低 +0.1%（今日上涨），并将“成交量”筛选为最低 500,000。

## 4. 手动调整指南
您可以手动更改筛选条件以适应您的风格：
- 降低接近度上限至30%，以寻找从更深底部反弹的股票。
- 将涨跌幅最小值提高到3%，以仅查看强劲反弹的股票。
- 将市值设置为最低5,000M，以关注破产风险较低的公司。
- 添加市盈率（P/E）筛选，以区分盈利公司。

## 5. 真实世界场景
免责声明：以下情景仅使用纯粹假设性的虚构数据，仅供说明之用。它并不反映真实的市场状况、实际的价格波动或真实的交易历史记录。假设情景：如果一家超大盘科技股较其 52 周峰值下跌了 45%，但在利好催化剂的作用下在单日交易中上涨了 4%，Bottom Bounce 会将其作为值得调查的潜在早期反转候选股票筛选出来。

## 6. 风险与警告
- ⚠️ 看起来像底部反转的情况可能是“死猫反弹”。
- ⚠️ 不要仅依赖技术信号，而不进行基本面分析。
- ⚠️ 许多这些股票下跌是有充分理由的——请检查公司的财务状况。
