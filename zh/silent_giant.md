# 沉默巨头
*为寻求避风港的投资者提供巨型市值堡垒*

## 1. 什么是此策略？
- **风险等级**: 低
- **Timeframe**: (3-12)
- **Target**: 保守投资者, 退休投资组合

沉默巨头策略寻找市值超过$1000亿且日内价格变动在-2%至+2%之间的股票。这些是受市场波动影响最小的稳定巨型市值公司。

## 2. 自动筛选参数
当您点击此卡片时，将自动应用以下筛选器：

| 列 | 应用值 |
| :--- | :--- |
| 实时成交量 | > 1.000.000 |
| 市值 | > 10B |

## 3. 工作原理？
市值最低 100,000M，涨跌幅筛选在 -2% 至 +2% 之间。结果按市值降序排列。

## 4. 手动调整指南
您可以手动更改筛选条件以适应您的风格：
- 将市值最低值提高到500,000M，以仅查看全球前20大公司。
- 将涨跌幅范围设置为-1%至+1%，以筛选出极其平静的巨头。
- 添加最高市盈率（P/E）25的筛选条件，以排除估值过高的科技巨头。
- 将板块筛选器设置为“非耐用消费品”以查看防御型巨头。

## 5. 真实世界场景
免责声明：以下情景仅出于演示目的，使用纯粹假设和虚构的数据。它并不反映真实的市场状况、实际的价格波动或真实的历史记录。假设情景：如果大盘下跌 3%，而一家超大盘医疗保健或消费必需品公司仅下跌 0.3%，Silent Giants 会将该股票标记为表现出相对强劲势头的潜在避风港。

## 6. 风险与警告
- ⚠️ 低波动性 = 低短期回报潜力。
- ⚠️ 即使是超大市值公司也可能受到行业危机的影响。
- ⚠️ 将此视为长期策略——不适合日内交易。
