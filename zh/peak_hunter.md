# 高点猎手
*发现准备突破 52 周高点的股票*

## 1. 什么是此策略？
- **风险等级**: 高
- **Timeframe**: (1-15)
- **Target**: 突破交易者, 动量交易者

高点猎手策略寻找交易价格高于其52周高点95%的股票。这些股票是“突破”的潜在标的。在技术分析中，突破52周高点被视为一个强烈的看涨信号。

## 2. 自动筛选参数
当您点击此卡片时，将自动应用以下筛选器：

| 列 | 应用值 |
| :--- | :--- |
| 实时成交量 | > 1.000.000 |
| 市值 | > 10B |

## 3. 工作原理？
在筛选器中，将“接近峰值”列筛选为最低 95%。结果按接近峰值的程度降序排列。

## 4. 手动调整指南
您可以手动更改筛选条件以适应您的风格：
- 将接近度阈值提高到98%，以仅查看处于突破边缘的股票。
- 添加至少500,000的成交量，以消除低流动性噪音。
- 将市值设置为最低1,000M，以过滤掉小型公司的噪音。
- 添加至少+0.5的涨跌幅（%），以仅查看今日上涨的候选股票。

## 5. 真实世界场景
免责声明：以下情景仅使用纯粹假设性的虚构数据，仅供说明之用。它并不反映真实的市场状况、实际的价格波动或真实的交易历史记录。假设情景：如果一家大盘股的交易价格处于其 52 周峰值的 97%，且成交量高于平均水平，Peak Hunter 会将其标记为接近潜在突破区域的候选股票。

## 6. 风险与警告
- ⚠️ 接近高点的股票可能是“多头陷阱”。
- ⚠️ 在整体市场下跌期间，个股突破往往会失败。
- ⚠️ 务必设置止损——高点拒绝可能导致急剧下跌。
