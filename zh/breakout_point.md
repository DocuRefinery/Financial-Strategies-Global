# 突破点
*实时检测经典的突破形态*

## 1. 什么是此策略？
- **风险等级**: 高
- **Timeframe**: (1-5)
- **Target**: 技术分析师, 突破交易者

突破点策略结合了三个关键筛选条件：股价接近峰值90%以上、成交量超过100万、以及日涨幅为正。这种组合是技术分析中经典“突破”的信号。

## 2. 自动筛选参数
当您点击此卡片时，将自动应用以下筛选器：

| 列 | 应用值 |
| :--- | :--- |
| 实时成交量 | > 1.000.000 |
| 市值 | > 10B |

## 3. 工作原理？
接近峰值最低 90%，成交量最低 1,000,000，涨跌幅最低 +0.1%。结果按接近峰值的程度降序排列。

## 4. 手动调整指南
您可以手动更改筛选条件以适应您的风格：
- 将接近峰值百分比提高到95%，以查看最成熟的突破候选股。
- 将最小成交量提高到200万，以增强机构确认。
- 将最小涨幅提高到1%，以查看哪些股票正以强劲势头突破。
- 添加市值最低5,000M，以关注大型公司的突破。

## 5. 真实世界场景
免责声明：以下情景仅使用纯粹假设性的虚构数据进行说明。它并不反映真实的市场状况、实际的价格波动或真实的交易历史记录。假设情景：如果某只超大市值电子商务股票处于其峰值的 93% 附近，日内上涨 1.8% 且成交量达到 8M，则将满足 Breakout Point 的所有三项标准——从而标记出一个经典的突破形态，以供进一步的技术分析。

## 6. 风险与警告
- ⚠️ 并非所有突破信号都会转化为真正的突破——假突破很常见。
- ⚠️ 检查大盘整体方向（SPY, QQQ）。
- ⚠️ 如果临近财报发布日期，请谨慎——波动性会增加。
