# 价值宝藏
*寻找被市场忽视的廉价巨头公司*

## 1. 什么是此策略？
- **风险等级**: 低
- **Timeframe**: (1-12)
- **Target**: ,

价值宝藏策略寻找市值大但市盈率（P/E）低的上市公司。这是沃伦·巴菲特式价值投资的数字化版本。发现那些基本面强劲却被市场忽视的巨头。

## 2. 自动筛选参数
当您点击此卡片时，将自动应用以下筛选器：

| 列 | 应用值 |
| :--- | :--- |
| 实时成交量 | > 1.000.000 |
| 市值 | > 10B |

## 3. 工作原理？
将市盈率筛选为最高 12 倍，并将市值设置为最低 10,000M（100 亿美元）。结果按市盈率升序排列。

## 4. 手动调整指南
您可以手动更改筛选条件以适应您的风格：
- 降低市盈率（P/E）上限至8，以查看超低估值的巨头。
- 将市值最小值提高到50,000M，以仅关注大型股。
- 将板块筛选设置为“金融”，以在银行业寻找价值机会。
- 将涨跌幅筛选设置为最低-2%和最高+2%，以查看平静稳定的巨头。

## 5. 真实世界场景
免责声明：以下情景仅使用纯粹假设性的虚构数据，仅供说明之用。它并不反映真实的市场状况、实际的价格波动或真实的交易历史记录。假设情景：如果一家多元化综合企业的 P/E 低于 10，且市值超过 $500B，Value Treasure 会在市场关注点转向他处时，将其标记为潜在的被低估巨头。

## 6. 风险与警告
- ⚠️ 低市盈率（P/E）不一定意味着“便宜”——请与行业平均水平进行比较。
- ⚠️ 有些公司市盈率（P/E）低是有原因的（停滞、债务等）。
- ⚠️ 不要忽视资产负债表分析，以避免“价值陷阱”。
