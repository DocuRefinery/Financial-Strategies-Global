# Ponto de Breakout
*Detecte a formação clássica de breakout ao vivo*

## 1. O que é esta Estratégia?
- **Nível de Risco**: Alto
- **Timeframe**: Curto Prazo (1-5 dias)
- **Target**: Analista Técnico, Breakout Trader

Breakout Point combina três filtros críticos: proximidade de 90%+ do pico, volume de 1M+ e variação diária positiva. Esta combinação é o sinal de alerta para um "breakout" clássico na análise técnica.

## 2. Parâmetros de Auto-Filtro
Ao clicar neste cartão, os seguintes filtros são aplicados automaticamente:

| Coluna | Valor Aplicado |
| :--- | :--- |
| Volume em Tempo Real | > 1.000.000 |
| Capitalização de Mercado | > 10B |

## 3. Como Funciona?
Proximidade do Pico mín 90%, Volume mín 1.000.000 e Variação mín +0,1%. Resultados listados em ordem decrescente por proximidade.

## 4. Guia de Ajuste Manual
Você pode alterar manualmente os filtros para se adequarem ao seu estilo:
- Aumente a proximidade para 95% para ver os candidatos a breakout mais maduros.
- Aumente o volume mínimo para 2M para fortalecer a confirmação institucional.
- Aumente a variação mínima para 1% para ver quem está em breakout com forte momentum.
- Adicione um Market Cap mínimo de 5.000M para focar em breakouts em empresas maiores.

## 5. Cenário do Mundo Real
AVISO LEGAL: O cenário a seguir utiliza dados puramente hipotéticos e fabricados apenas para fins ilustrativos. Ele NÃO reflete as condições reais do mercado, movimentos de preços reais ou registros históricos verdadeiros. Cenário hipotético: Se uma ação de e-commerce mega-cap estivesse 93% próxima de seu pico, subisse 1,8% no intraday com volume de 8M, todos os três critérios do Breakout Point seriam atendidos — sinalizando uma formação clássica de breakout para posterior análise técnica.

## 6. Riscos e Avisos
- ⚠️ Nem todo sinal de breakout se transforma em um breakout real — fakeouts são comuns.
- ⚠️ Verifique a direção geral do mercado (SPY, QQQ).
- ⚠️ Tenha cuidado se uma data de balanço estiver próxima — a volatilidade aumenta.
