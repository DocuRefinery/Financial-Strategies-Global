# Tesouro de Valor
*Encontre empresas gigantes baratas que o mercado ignorou*

## 1. O que é esta Estratégia?
- **Nível de Risco**: Baixo
- **Timeframe**: Longo Prazo (1-12 meses)
- **Target**: Value Investor, Investidor de Longo Prazo

O Value Treasure encontra empresas de grande capitalização de mercado com baixos índices Price/Earnings (P/E). É uma versão digital do estilo de value investing de Warren Buffett. Descubra gigantes com fundamentos sólidos ignorados pelo mercado.

## 2. Parâmetros de Auto-Filtro
Ao clicar neste cartão, os seguintes filtros são aplicados automaticamente:

| Coluna | Valor Aplicado |
| :--- | :--- |
| Volume em Tempo Real | > 1.000.000 |
| Capitalização de Mercado | > 10B |

## 3. Como Funciona?
O P/E ratio é filtrado em um máximo de 12 e o Market Cap é definido para um mínimo de 10.000M ($10 bilhões). Os resultados são listados em ordem crescente de P/E.

## 4. Guia de Ajuste Manual
Você pode alterar manualmente os filtros para se adequarem ao seu estilo:
- Reduza o limite superior de P/E para 8 para ver gigantes ultra-baratas.
- Aumente o Market Cap mínimo para 50.000M para focar apenas em empresas mega-cap.
- Defina o filtro de setor para "Finance" para encontrar oportunidades de valor no setor bancário.
- Defina o filtro de Change para -2 mín e +2 máx para ver gigantes calmas e estáveis.

## 5. Cenário do Mundo Real
AVISO LEGAL: O cenário a seguir utiliza dados puramente hipotéticos e fabricados apenas para fins ilustrativos. NÃO reflete condições reais de mercado, movimentos de preços reais ou registros históricos verdadeiros. Cenário hipotético: Se um conglomerado diversificado tivesse um P/E ratio abaixo de 10 com um market cap superior a $500B, o Value Treasure a sinalizaria como uma gigante potencialmente subvalorizada enquanto o mercado em geral foca em outros lugares.

## 6. Riscos e Avisos
- ⚠️ P/E baixo nem sempre significa "barato" — compare com a média do setor.
- ⚠️ Algumas empresas têm P/E baixo por bons motivos (estagnação, dívida, etc.).
- ⚠️ Não negligencie a análise do balanço patrimonial para evitar "value traps."
