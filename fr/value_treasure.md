# Trésor de Valeur
*Trouver les géants sous-évalués que le marché ignore*

## 1. Quelle est cette Stratégie ?
- **Niveau de Risque**: Faible
- **Timeframe**: Long terme (1-12 mois)
- **Target**: Investisseur Value, Long-termiste

Trésor de Valeur trouve les entreprises à grande capitalisation avec un PER bas. La version numérique de l'investissement value style Warren Buffett. Découvrez les géants aux fondamentaux solides ignorés par le marché.

## 2. Paramètres de Filtre Auto
Lorsque vous cliquez sur cette carte, les filtres suivants sont automatiquement appliqués :

| Colonne | Valeur Appliquée |
| :--- | :--- |
| Volume en Direct | > 1.000.000 |
| Cap. Boursière | > 10B |

## 3. Comment ça marche ?
PER maximum 12, Capitalisation minimum 10 000M (10 Mrd $). Résultats classés par PER croissant.

## 4. Guide d'Ajustement Manuel
Vous pouvez modifier manuellement les filtres pour les adapter à votre style :
- Abaissez le PER à 8 pour voir les géants ultra-décotés.
- Augmentez la capitalisation à 50 000M pour uniquement les méga-caps.
- Filtrez le secteur "Finance" pour les opportunités value bancaires.
- Fixez la variation entre -2 et +2 pour voir les géants calmes et stables.

## 5. Scénario Réel
AVERTISSEMENT : Le scénario suivant utilise des données purement hypothétiques et fictives à des fins d'illustration uniquement. Il ne reflète PAS les conditions réelles du marché, les mouvements de prix réels ou les données historiques véridiques. Scénario hypothétique : Si un conglomérat diversifié affichait un ratio P/E inférieur à 10 avec une capitalisation boursière dépassant 500 milliards de dollars, Value Treasure le signalerait comme un géant potentiellement sous-évalué pendant que le reste du marché se concentre ailleurs.

## 6. Risques & Avertissements
- ⚠️ Un PER bas ne signifie pas toujours "bon marché" — comparez avec la moyenne du secteur.
- ⚠️ Certaines entreprises ont un PER bas pour de bonnes raisons (stagnation, dette, etc.).
- ⚠️ N'oubliez pas l'analyse du bilan pour éviter les "pièges de valeur".
