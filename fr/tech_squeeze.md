# Compression Tech
*Capturer les signaux de compression et rallye dans le secteur tech*

## 1. Quelle est cette Stratégie ?
- **Niveau de Risque**: Moyen-Élevé
- **Timeframe**: Court-Moyen terme (3-15 jours)
- **Target**: Rotateur Sectoriel, Suiveur Tech

Compression Tech filtre les actions "Technology Services" par volume élevé et proximité au sommet. Détecte les périodes où le secteur entier se compresse et se prépare à un rallye.

## 2. Paramètres de Filtre Auto
Lorsque vous cliquez sur cette carte, les filtres suivants sont automatiquement appliqués :

| Colonne | Valeur Appliquée |
| :--- | :--- |
| Volume en Direct | > 1.000.000 |
| Cap. Boursière | > 10B |

## 3. Comment ça marche ?
Secteur "Technology Services", Volume min. 1 000 000, Variation min. 0%, Proximité min. 85%. Par volume décroissant.

## 4. Guide d'Ajustement Manuel
Vous pouvez modifier manuellement les filtres pour les adapter à votre style :
- Passez à "Electronic Technology" pour le matériel et semi-conducteurs.
- Proximité à 95% pour les candidats breakout leaders.
- Volume à 5M pour uniquement les méga-volumes tech.
- PER max. 40 pour exclure les growth surévaluées.

## 5. Scénario Réel
AVERTISSEMENT : Le scénario suivant utilise des données purement hypothétiques et fictives à des fins d'illustration uniquement. Il ne reflète PAS les conditions réelles du marché, les mouvements de prix réels ou les données historiques véridiques. Scénario hypothétique : Si plusieurs actions technologiques à méga-capitalisation se négociaient toutes à plus de 90 % de leurs sommets avec un volume élevé avant un événement macroéconomique, Tech Squeeze signalerait ce modèle de regroupement comme une configuration potentielle à l'échelle du secteur méritant d'être surveillée.

## 6. Risques & Avertissements
- ⚠️ Les compressions sectorielles sont très influencées par les facteurs macro.
- ⚠️ Les décisions de la FED et l'inflation affectent le tech de manière disproportionnée.
- ⚠️ Trop de concentration sectorielle nuit à la diversification.
