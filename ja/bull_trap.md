# ブルトラップ・スキャナー
*偽のブレイクアウトと高値での拒絶シグナルを検出します*

## 1. この戦略とは何ですか？
- **リスクレベル**: 高
- **Timeframe**: 15
- **Target**: 空売り投資家、テクニカルアナリスト

ブルトラップ・スキャナーは、高値の95%を超えているものの、本日は2%以上下落している銘柄を検出します。これは典型的な「ブルトラップ」の形成であり、高値付近の銘柄が拒絶され、売り圧力が始まったことを示しています。ショートポジションや利益確定のシグナルとなる可能性があります。

## 2. 自動フィルターパラメータ
このカードをクリックすると、以下のフィルターが自動的に適用されます：

| カラム | 適用値 |
| :--- | :--- |
| ライブ出来高 | > 1.000.000 |
| 時価総額 | > 10B |

## 3. 仕組み
高値乖離率 95%以上、変化率 -2%以下、出来高 1,000,000以上。結果は変化率の昇順（下落幅が大きい順）で表示されます。

## 4. マニュアルチューニングガイド
スタイルに合わせてフィルターを手動で変更できます：
- 変化率の最大値を-4%に下げ、より強力な拒絶シグナルを表示します。
- 出来高の最小値を2Mに引き上げ、機関投資家による売り圧力を確認します。
- 時価総額の最小値を5,000Mに設定し、大型株におけるブルトラップに焦点を当てます。
- 近接度を98%に引き上げ、正確なピーク拒絶を表示します。

## 5. 実例シナリオ
免責事項：以下のシナリオは、説明のみを目的とした純粋に仮定の架空のデータを使用しています。実際の市場状況、実際の価格変動、または真の過去の記録を反映するものではありません。仮定のシナリオ：大手エンタープライズSaaS企業がピークの97%に達した後、大量の出来高を伴って日中に3.5%下落した場合、Bull Trap Scannerはこのパターンを典型的なピーク拒絶（peak rejection）としてフラグを立て、反転の可能性を示唆します。

## 6. リスクと警告
- ⚠️ すべてのピーク拒絶が恒久的な反転とは限りません。一時的な調整である可能性もあります。
- ⚠️ 空売りには無限の損失リスクが伴います。必ずストップロスを設定してください。
- ⚠️ 決算発表後の下落は異なる力学で動きます。ニュースを確認してください。
