# バリュートレジャー
*市場が見落としている割安な巨大企業を見つける*

## 1. この戦略とは何ですか？
- **リスクレベル**: 低
- **Timeframe**: 112
- **Target**: バリュー投資家、長期投資家

Value Treasureは、株価収益率（P/E）が低い時価総額の大きな企業を見つけます。これはウォーレン・バフェット流のバリュー投資をデジタル化したものです。市場に無視されている、ファンダメンタルズの強い巨人を発掘しましょう。

## 2. 自動フィルターパラメータ
このカードをクリックすると、以下のフィルターが自動的に適用されます：

| カラム | 適用値 |
| :--- | :--- |
| ライブ出来高 | > 1.000.000 |
| 時価総額 | > 10B |

## 3. 仕組み
P/E比率を最大12でフィルタリングし、時価総額を最小10,000M（100億ドル）に設定します。結果はP/Eの昇順で表示されます。

## 4. マニュアルチューニングガイド
スタイルに合わせてフィルターを手動で変更できます：
- P/Eの上限を8に下げると、超割安な巨大企業を見つけることができます。
- 時価総額の最小値を50,000Mに引き上げると、メガキャップ企業のみに絞り込むことができます。
- セクターフィルターを「金融」に設定すると、銀行業界のバリュー投資の機会を見つけることができます。
- 変動率フィルターを最小-2、最大+2に設定すると、穏やかで安定した巨大企業を見ることができます。

## 5. 実例シナリオ
免責事項：以下のシナリオは、説明のみを目的とした純粋に仮定の架空のデータを使用しています。実際の市場状況、実際の価格変動、または真の過去の記録を反映するものではありません。仮定のシナリオ：もし多角経営のコングロマリットのP/Eが10未満で、時価総額が5,000億ドルを超えている場合、市場全体が他の場所に注目している間に、Value Treasureはそれを潜在的に割安な巨大企業としてフラグを立てます。

## 6. リスクと警告
- ⚠️ P/Eが低いことが必ずしも「割安」を意味するわけではありません。セクター平均と比較してください。
- ⚠️ 一部の企業では、停滞や負債などの正当な理由でP/Eが低くなっている場合があります。
- ⚠️ 「バリュートラップ（割安の罠）」を避けるため、貸借対照表の分析を怠らないようにしてください。
