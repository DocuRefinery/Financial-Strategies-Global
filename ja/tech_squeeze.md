# テックスクイーズ
*テックセクターにおけるスクイーズとラリーのシグナルを捉える*

## 1. この戦略とは何ですか？
- **リスクレベル**: 中〜高
- **Timeframe**: (315)
- **Target**: セクターローテーター、テックフォロワー

Tech Squeezeは、「テクノロジーサービス」セクターの銘柄を、高い出来高とピークへの近さでフィルタリングします。これは、セクター全体がスクイーズし、ラリーの準備をしている期間を検出するように設計されています。

## 2. 自動フィルターパラメータ
このカードをクリックすると、以下のフィルターが自動的に適用されます：

| カラム | 適用値 |
| :--- | :--- |
| ライブ出来高 | > 1.000.000 |
| 時価総額 | > 10B |

## 3. 仕組み
セクターが「Technology Services」に絞られ、出来高が最低1,000,000、変化率が最低0%、高値からの近接度が最低85%。結果は出来高の降順で表示されます。

## 4. マニュアルチューニングガイド
スタイルに合わせてフィルターを手動で変更できます：
- セクターを「電子技術」に変更して、ハードウェアおよび半導体企業をスキャンします。
- セクターリーダーからのブレイクアウト候補を見つけるには、近接度（proximity）を95%に引き上げてください。
- メガボリュームのハイテク株のみを追跡するには、最小出来高を5Mに引き上げてください。
- 過大評価されているグロース株を除外するには、最大P/Eを40に設定してください。

## 5. 実例シナリオ
免責事項：以下のシナリオは、説明のみを目的とした純粋に仮定の捏造されたデータを使用しています。これは実際の市場状況、実際の価格変動、または真の過去の記録を反映するものではありません。仮定のシナリオ：マクロイベントを前に、複数のメガキャップ・ハイテク株がすべてピークの90%以上で取引され、出来高が増加している場合、Tech Squeezeはこのクラスターパターンを、監視に値するセクター全体の潜在的なセットアップとしてフラグを立てます。

## 6. リスクと警告
- ⚠️ セクターのスクイーズは、マクロ経済要因の影響を強く受けます。
- ⚠️ FRBの決定やインフレデータは、ハイテクセクターに不釣り合いなほど大きな影響を与えます。
- ⚠️ 特定のセクターに集中しすぎると、ポートフォリオの分散効果が損なわれます。
