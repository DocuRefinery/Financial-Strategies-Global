# 割安なブルーチップ
*割安な価格で大手企業を捉える*

## 1. この戦略とは何ですか？
- **リスクレベル**: 低〜中
- **Timeframe**: (1-6)
- **Target**: バリュー投資家、長期投資家

Discounted Blue Chipは、時価総額が200億ドルを超え、ピークから35%以上下落しており、妥当なP/Eで取引されている企業を見つけ出します。これらの巨大企業は、一時的な問題により割安になっている可能性があり、質の高い資産を割引価格で購入するための理想的な戦略です。

## 2. 自動フィルターパラメータ
このカードをクリックすると、以下のフィルターが自動的に適用されます：

| カラム | 適用値 |
| :--- | :--- |
| ライブ出来高 | > 1.000.000 |
| 時価総額 | > 10B |

## 3. 仕組み
時価総額が最低20,000M、高値からの近接度が最大65%、かつP/Eレシオが0.1〜20の間。結果は時価総額の降順で表示されます。

## 4. マニュアルチューニングガイド
スタイルに合わせてフィルターを手動で変更できます：
- 近接度（Proximity）の最大値を50%に下げると、より深く割安な銘柄を表示できます。
- 最大P/Eを12に下げると、極めて割安な優良株を絞り込めます。
- 変化率（Change）の最小値を0に追加すると、本日回復傾向にある割安株を表示できます。
- 時価総額（Market Cap）の最小値を50,000Mに引き上げると、超大型株の割安銘柄に集中できます。

## 5. 実例シナリオ
免責事項：以下のシナリオは、説明のみを目的とした純粋に仮定の架空のデータを使用しています。実際の市場状況、実際の価格変動、または真の過去の記録を反映するものではありません。仮定のシナリオ：世界的なアスレチックブランド（時価総額約1,200億ドル、P/E 18）が、ブランドの堀（moat）は維持されているものの、消費者信頼感への懸念からピークから40%下落した場合、Discounted Blue Chipは、それを過度に割安な優良銘柄としてリストアップします。

## 6. リスクと警告
- ⚠️ 割安であることは必ずしも好機とは限りません。企業の競争上の地位が損なわれていないか確認してください。
- ⚠️ 回復には時間がかかる場合があります。少なくとも6〜12ヶ月の期間を想定して計画してください。
- ⚠️ 構造的なセクターの変化（破壊的イノベーションなど）により、古い優良株が恒久的に割安になる可能性があります。
