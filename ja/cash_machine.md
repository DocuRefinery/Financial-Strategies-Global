# キャッシュ・マシン
*安定した収益性で定期的なキャッシュを生み出す巨大企業を見つけます*

## 1. この戦略とは何ですか？
- **リスクレベル**: 低
- **Timeframe**: 624
- **Target**: インカム投資家、リタイアメントプランナー

キャッシュ・マシンは、P/Eが5〜15（強力な収益性）、時価総額300億ドル以上、安定した価格変動（-1%から+1%）、および高値から50〜80%の近接性を持つ企業を見つけます。このプロファイルは、定期的なキャッシュフローを生み出し、過大評価されていない安定した巨大企業を指し示しています。

## 2. 自動フィルターパラメータ
このカードをクリックすると、以下のフィルターが自動的に適用されます：

| カラム | 適用値 |
| :--- | :--- |
| ライブ出来高 | > 1.000.000 |
| 時価総額 | > 10B |

## 3. 仕組み
P/E 5-15、時価総額 30,000M以上、変化率 -1%から+1%、高値乖離率 50-80%。結果はP/Eの昇順で表示されます。

## 4. マニュアルチューニングガイド
スタイルに合わせてフィルターを手動で変更できます：
- 最大P/Eを10に下げると、最も収益性の高い巨大企業が表示されます。
- 時価総額の最小値を100,000Mに上げると、メガキャップのキャッシュマシンに焦点を絞れます。
- 範囲を-0.5%から+0.5%に変更すると、極めて安定した銘柄を抽出できます。
- セクターを「公益事業 (Utilities)」に設定すると、インフラ系のキャッシュマシンが表示されます。

## 5. 実例シナリオ
免責事項：以下のシナリオは、説明のみを目的とした純粋に仮定の架空のデータを使用しています。実際の市場状況、実際の価格変動、または真の過去の記録を反映するものではありません。仮定のシナリオ：大型通信会社（時価総額約1,700億ドル、P/E 9）が、ピークを大幅に下回る水準で取引されながらも、安定した値動き、強力なキャッシュフロー、定期的な配当を維持している場合、Cash Machineは、それを着実なリターンを生み出す「退屈だが収益性の高い」巨大企業の一つとしてリストアップします。

## 6. リスクと警告
- ⚠️ 低いP/Eと大きな時価総額は、成長の停滞を意味する可能性があります。
- ⚠️ 安定した株価の動きは、その銘柄への関心の欠如である可能性もあります。
- ⚠️ セクターの破壊的リスク（技術革新など）が、長期的な最大の脅威です。
