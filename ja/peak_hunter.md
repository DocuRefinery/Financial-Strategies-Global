# ピークハンター
*52週高値の更新を控えた銘柄を見つける*

## 1. この戦略とは何ですか？
- **リスクレベル**: 高
- **Timeframe**: 115
- **Target**: ブレイクアウトトレーダー、モメンタムトレーダー

Peak Hunterは、52週高値の95%以上で取引されている銘柄を見つけます。これらの銘柄は「ブレイクアウト」の候補となります。テクニカル分析において、52週高値の更新は強力な強気シグナルとみなされます。

## 2. 自動フィルターパラメータ
このカードをクリックすると、以下のフィルターが自動的に適用されます：

| カラム | 適用値 |
| :--- | :--- |
| ライブ出来高 | > 1.000.000 |
| 時価総額 | > 10B |

## 3. 仕組み
スクリーナーの「ピークへの近接度」列を最小95%でフィルタリングします。結果は近接度の降順で表示されます。

## 4. マニュアルチューニングガイド
スタイルに合わせてフィルターを手動で変更できます：
- 高値乖離率のしきい値を98%に引き上げると、ブレイクアウト直前の銘柄のみを表示できます。
- 流動性の低い銘柄のノイズを除去するため、出来高を最低500,000以上に設定してください。
- 小型株のノイズを除去するため、時価総額を最低1,000M以上に設定してください。
- 本日上昇している銘柄のみを表示するため、変化率(%)に最低+0.5を追加してください。

## 5. 実例シナリオ
免責事項：以下のシナリオは、説明のみを目的とした純粋に仮定の架空のデータを使用しています。実際の市場状況、実際の価格変動、または真の過去の記録を反映するものではありません。仮定のシナリオ：もし大型株が平均以上の出来高を伴い、52週高値の97%で取引されている場合、Peak Hunterはそれを潜在的なブレイクアウトゾーンに近づいている候補としてフラグを立てます。

## 6. リスクと警告
- ⚠️ 高値圏にある銘柄は「ブルトラップ（買いの罠）」となる可能性があります。
- ⚠️ 市場全体が下落している局面では、個別のブレイクアウトは失敗することがよくあります。
- ⚠️ 高値からの反落は急落につながる可能性があるため、必ずストップロスを設定してください。
