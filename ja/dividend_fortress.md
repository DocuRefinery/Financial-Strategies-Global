# ディビデンド・フォートレス
*定期的なインカムを求める投資家のための、高い配当ポテンシャルを持つ銘柄*

## 1. この戦略とは何ですか？
- **リスクレベル**: 低
- **Timeframe**: (624)
- **Target**: 配当投資家、不労所得追求者

ディビデンド・フォートレスは、時価総額500億ドル以上、P/E比率18未満の企業を見つけ出します。このプロファイルに該当する企業は、通常、定期的な配当を支払う成熟した収益性の高い巨大企業です。

## 2. 自動フィルターパラメータ
このカードをクリックすると、以下のフィルターが自動的に適用されます：

| カラム | 適用値 |
| :--- | :--- |
| ライブ出来高 | > 1.000.000 |
| 時価総額 | > 10B |

## 3. 仕組み
時価総額が最低50,000MかつP/Eレシオが0.1〜18の間。結果は時価総額の降順で表示されます。

## 4. マニュアルチューニングガイド
スタイルに合わせてフィルターを手動で変更できます：
- P/Eの上限を12に下げると、より割安な配当銘柄候補が表示されます。
- 時価総額の最小値を100,000Mに引き上げると、メガキャップの配当大手銘柄のみが表示されます。
- セクターを「公益事業」または「金融」に設定して、典型的な高配当セクターをスキャンします。
- 変化率（Change）フィルターを-1% / +1%に設定すると、値動きが穏やかな大手銘柄が表示されます。

## 5. 実例シナリオ
免責事項：以下のシナリオは、説明のみを目的とした純粋に仮定の捏造されたデータを使用しています。これは実際の市場状況、実際の価格変動、または真の過去の記録を反映するものではありません。仮定のシナリオ：時価総額3,500億ドル、P/Eが13の生活必需品大手が数十年にわたって配当を増やし続けてきた場合、Dividend Fortressはそれを「配当貴族（Dividend Aristocrat）」のプロフィールに合致する候補として提示します。

## 6. リスクと警告
- ⚠️ このスクリーナーには配当利回りは表示されません。配当履歴は別途調査してください。
- ⚠️ 低いP/Eと大きな時価総額が、必ずしも高い配当を意味するわけではありません。
- ⚠️ 配当性向を確認し、減配リスクのある企業を見極めてください。
