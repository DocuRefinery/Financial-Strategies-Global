# メガ・サプライズ
*巨大企業における異例の大幅な動きを捉える*

## 1. この戦略とは何ですか？
- **リスクレベル**: 中
- **Timeframe**: (1-5)
- **Target**: ニュースベースのトレーダー、イベント・フォロワー

Mega Surprise（メガ・サプライズ）は、時価総額500億ドルを超える巨大企業における3%以上の1日の価格変動を特定します。メガキャップ銘柄は通常0.5〜1%程度しか動きません。3%以上の動きは、決算、合併、規制、またはセクター全体の激震を意味します。

## 2. 自動フィルターパラメータ
このカードをクリックすると、以下のフィルターが自動的に適用されます：

| カラム | 適用値 |
| :--- | :--- |
| ライブ出来高 | > 1.000.000 |
| 時価総額 | > 10B |

## 3. 仕組み
時価総額が最低50,000Mかつ変化率が最低+3%。結果は変化率の降順で表示されます。

## 4. マニュアルチューニングガイド
スタイルに合わせてフィルターを手動で変更できます：
- 変動率の最小値を5%に引き上げると、「ショック」レベルのメガムーブのみを表示できます。
- 時価総額の最小値を200,000Mに引き上げると、世界最大級の企業のサプライズのみを表示できます。
- 出来高フィルターを追加して、その動きが出来高を伴っているか確認してください。
- セクターフィルターを追加して、特定のセクター内のメガサプライズに注目してください。

## 5. 実例シナリオ
免責事項：以下のシナリオは、説明のみを目的とした純粋に仮定の架空のデータを使用しています。実際の市場状況、実際の価格変動、または真の過去の記録を反映するものではありません。仮定のシナリオ：時価総額1兆ドル以上の企業が、予想を大幅に上回る決算を発表した後、1回のセッションで5％急騰した場合、Mega Surpriseはこの異例の動きを即座に検出します。巨大企業の動きは、しばしばより広範なセクタートレンドを示唆することがあるためです。

## 6. リスクと警告
- ⚠️ メガキャップのサプライズは決算に関連していることが多く、モメンタムが短期間で終わる可能性があります。
- ⚠️ 大幅な上昇の後にエントリーすることは、「高値掴み」のリスクを伴います。
- ⚠️ サプライズは下落方向にも起こり得ます。一方的な見方をしないようにしてください。
