# ペニー・ハンター
*超低価格株における爆発的なチャンスを発見する*

## 1. この戦略とは何ですか？
- **リスクレベル**: 非常に高い
- **Timeframe**: (1-3)
- **Target**: 投機家、ハイリスク・トレーダー

Penny Hunter（ペニー・ハンター）は、価格が5ドル未満、上昇率が5%以上、出来高が100万を超える銘柄を特定します。ペニー株は最もリスクが高いですが、最大の潜在的リターン（または損失）を提供します。この戦略は、高いリスク許容度を持つ経験豊富なトレーダーのみを対象としています。

## 2. 自動フィルターパラメータ
このカードをクリックすると、以下のフィルターが自動的に適用されます：

| カラム | 適用値 |
| :--- | :--- |
| ライブ出来高 | > 1.000.000 |
| 時価総額 | > 10B |

## 3. 仕組み
価格が最大$5、変化率が最低+5%、かつ出来高が最低1,000,000。結果は変化率の降順で表示されます。

## 4. マニュアルチューニングガイド
スタイルに合わせてフィルターを手動で変更できます：
- 価格上限を3ドルに下げると、超低位のペニー株を表示できます。
- 変動率の最小値を10%に引き上げると、最も爆発的な動きのみを表示できます。
- 出来高の最小値を3Mに引き上げると、より信頼性の高い流動性を持つペニー株に絞り込めます。
- 価格下限を1ドルに設定すると、1ドル未満の危険な銘柄を除外できます。

## 5. 実例シナリオ
免責事項：以下のシナリオは、説明のみを目的とした純粋に仮定の架空のデータを使用しています。実際の市場状況、実際の価格変動、または真の過去の記録を反映するものではありません。仮定のシナリオ：バイオテクノロジーセクターのペニーストック（2.50ドル前後）が、臨床試験の結果を受けて500万の出来高を伴い25％急騰した場合、Penny Hunterはこの爆発的な動きを即座に捉えます。このような取引には極めて高いリスクが伴います。

## 6. リスクと警告
- ⚠️ ペニー株は最も危険な投資カテゴリーであり、資本をすべて失う可能性があります。
- ⚠️ パンプ＆ダンプ、相場操縦、フェイクニュースが非常に一般的です。
- ⚠️ スプレッドが非常に広くなる可能性があり、買値と売値の差が利益を圧迫することがあります。
- ⚠️ ポートフォリオ全体の最大1〜2%の配分に留めてください。
