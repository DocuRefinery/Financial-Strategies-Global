# フォーリング・スター
*ピークから下落したかつてのリーダー銘柄に回復の機会を求める*

## 1. この戦略とは何ですか？
- **リスクレベル**: 高
- **Timeframe**: (15-90)
- **Target**: 逆張り投資家、バリューハンター

Falling Starは、時価総額が200億ドルを超え、52週高値から40%以上下落している銘柄を特定します。これらはかつての市場リーダーであり、問題が一時的であれば、大きなターンアラウンドの可能性を秘めています。

## 2. 自動フィルターパラメータ
このカードをクリックすると、以下のフィルターが自動的に適用されます：

| カラム | 適用値 |
| :--- | :--- |
| ライブ出来高 | > 1.000.000 |
| 時価総額 | > 10B |

## 3. 仕組み
時価総額が最低20,000Mかつ高値からの近接度が最大60%、出来高が最低1,000,000。結果は近接度の昇順（下落幅が大きい順）で表示されます。

## 4. マニュアルチューニングガイド
スタイルに合わせてフィルターを手動で変更できます：
- 時価総額の最小値を50,000Mに引き上げて、メガキャップの「落ちてきたスター」のみを表示します。
- 「下落率（Proximity）」の最大値を40%に下げると、より深い下落銘柄を見つけることができます。
- 「変化率（Change）」の最小値を+1に追加すると、本日回復の兆しを見せているフォーリングスターを確認できます。
- P/Eフィルターを追加して、依然として収益性のあるフォーリングスターを選別しましょう。

## 5. 実例シナリオ
免責事項：以下のシナリオは、説明のみを目的とした純粋に仮定の捏造されたデータを使用しています。これは実際の市場状況、実際の価格変動、または真の過去の記録を反映するものではありません。仮定のシナリオ：かつて急成長したフィンテック企業（時価総額約600億ドル）が、依然として多額の収益を上げながらピークから55%下落した場合、Falling Starは、より詳細なファンダメンタル分析に値する「かつての王者」の中にそれをリストアップします。

## 6. リスクと警告
- ⚠️ 大企業が下落するには理由があります。その下落が構造的なものか、循環的なものかを分析してください。
- ⚠️ 回復には数ヶ月から数年かかる場合があります。忍耐が必要です。
- ⚠️ 「バリュートラップ（割安の罠）」のリスクが高いです。企業の競争優位性が損なわれていないか確認してください。
