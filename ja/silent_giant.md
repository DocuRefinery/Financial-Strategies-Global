# サイレント・ジャイアンツ
*安全な避難所を求める投資家のためのメガキャップの要塞*

## 1. この戦略とは何ですか？
- **リスクレベル**: 低い
- **Timeframe**: (312)
- **Target**: 保守的な投資家、退職金ポートフォリオ

Silent Giantsは、時価総額が1,000億ドルを超え、1日の価格変動が-2%から+2%の間の銘柄を見つけ出します。これらは市場の変動の影響を最も受けにくい、安定したメガキャップ企業です。

## 2. 自動フィルターパラメータ
このカードをクリックすると、以下のフィルターが自動的に適用されます：

| カラム | 適用値 |
| :--- | :--- |
| ライブ出来高 | > 1.000.000 |
| 時価総額 | > 10B |

## 3. 仕組み
時価総額が最低100,000Mかつ変化率が-2%から+2%の間。結果は時価総額の降順で表示されます。

## 4. マニュアルチューニングガイド
スタイルに合わせてフィルターを手動で変更できます：
- 時価総額の最小値を500,000Mに引き上げ、世界トップ20社のみを表示しましょう。
- 変動率の範囲を-1%から+1%に設定し、極めて穏やかな巨大企業を抽出しましょう。
- P/Eの最大値を25に設定し、割高な巨大テック企業を除外しましょう。
- セクターフィルターを「Consumer Non-Durables（生活必需品）」に設定し、ディフェンシブな巨大企業を確認しましょう。

## 5. 実例シナリオ
免責事項：以下のシナリオは、説明のみを目的とした純粋に仮定の捏造されたデータを使用しています。これは実際の市場状況、実際の価格変動、または真の過去の記録を反映するものではありません。仮定のシナリオ：市場全体が3%下落する一方で、メガキャップのヘルスケア企業や生活必需品企業がわずか0.3%しか下落しなかった場合、Silent Giantsはこの銘柄を相対的な強さを示す潜在的なセーフハーバーとしてフラグを立てます。

## 6. リスクと警告
- ⚠️ 低ボラティリティは、短期間での高いリターンが見込めないことを意味します。
- ⚠️ メガキャップ銘柄であっても、セクター全体の危機には影響を受ける可能性があります。
- ⚠️ これは長期戦略としてお考えください。デイトレードには適していません。
