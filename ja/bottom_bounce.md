# ボトムバウンス
*V字回復を見せ、回復の兆しを見せている銘柄を捉える*

## 1. この戦略とは何ですか？
- **リスクレベル**: 高
- **Timeframe**: 530
- **Target**: バリュー投資家、逆張りトレーダー

Bottom Bounce戦略は、52週高値から50%以上下落したが、本日はプラスで引けた銘柄を特定します。これは典型的なV字回復の形成であり、底からの回復が始まった可能性があります。

## 2. 自動フィルターパラメータ
このカードをクリックすると、以下のフィルターが自動的に適用されます：

| カラム | 適用値 |
| :--- | :--- |
| ライブ出来高 | > 1.000.000 |
| 時価総額 | > 10B |

## 3. 仕組み
「ピークへの近接度」を最大50%（ピークから遠い）でフィルタリングし、「騰落率 (%)」を最小+0.1（本日はプラス）に設定し、「出来高」を最小500,000でフィルタリングします。

## 4. マニュアルチューニングガイド
スタイルに合わせてフィルターを手動で変更できます：
- より深い底からの反転銘柄を見つけるには、近接度の最大値を30%に下げてください。
- 力強い回復を見せている銘柄のみを表示するには、変化率の最小値を3%に上げてください。
- 倒産リスクの低い企業に絞るため、時価総額を最低5,000Mに設定してください。
- 収益性の高い企業を区別するために、P/Eフィルターを追加してください。

## 5. 実例シナリオ
免責事項：以下のシナリオは、説明のみを目的とした純粋に仮定の架空のデータを使用しています。実際の市場状況、実際の価格変動、または真の過去の記録を反映するものではありません。仮定のシナリオ：もしメガキャップのハイテク株が52週高値から45%下落しているものの、ポジティブなカタリストを受けて1回のセッションで4%上昇した場合、Bottom Bounceはそれを調査に値する潜在的な初期段階のターンアラウンド候補として浮上させます。

## 6. リスクと警告
- ⚠️ 底打ちに見える動きが「デッド・キャット・バウンス（死んだ猫の反発）」である可能性があります。
- ⚠️ ファンダメンタルズ分析を行わず、テクニカルシグナルのみに頼らないでください。
- ⚠️ これらの銘柄の多くは下落する正当な理由があるため、企業の財務健全性を確認してください。
