# ブレイクアウトポイント
*クラシックなブレイクアウト形成をリアルタイムで検出*

## 1. この戦略とは何ですか？
- **リスクレベル**: 高
- **Timeframe**: (15)
- **Target**: テクニカルアナリスト、ブレイクアウトトレーダー

ブレイクアウト・ポイントは、ピークへの90%以上の近接、100万以上の出来高、およびプラスの日次変化という3つの重要なフィルターを組み合わせています。この組み合わせは、テクニカル分析におけるクラシックな「ブレイクアウト」の合図となります。

## 2. 自動フィルターパラメータ
このカードをクリックすると、以下のフィルターが自動的に適用されます：

| カラム | 適用値 |
| :--- | :--- |
| ライブ出来高 | > 1.000.000 |
| 時価総額 | > 10B |

## 3. 仕組み
高値からの近接度が最低90%、出来高が最低1,000,000、かつ変化率が最低+0.1%。結果は近接度の降順で表示されます。

## 4. マニュアルチューニングガイド
スタイルに合わせてフィルターを手動で変更できます：
- 最も成熟したブレイクアウト候補を表示するには、近接度（proximity）を95%に引き上げてください。
- 機関投資家の確認を強化するには、最小出来高を2Mに引き上げてください。
- 強いモメンタムでブレイクアウトしている銘柄を表示するには、最小変化率を1%に引き上げてください。
- 大型企業のブレイクアウトに注目するには、最小時価総額を5,000Mに設定してください。

## 5. 実例シナリオ
免責事項：以下のシナリオは、説明のみを目的とした純粋に仮定の捏造されたデータを使用しています。これは実際の市場状況、実際の価格変動、または真の過去の記録を反映するものではありません。仮定のシナリオ：メガキャップのEコマース株がピークの93%付近にあり、日中に800万の出来高を伴って1.8%上昇した場合、Breakout Pointの3つの基準すべてが満たされ、さらなるテクニカル分析が必要な典型的なブレイクアウト形成としてフラグが立てられます。

## 6. リスクと警告
- ⚠️ すべてのブレイクアウトシグナルが本物のブレイクアウトにつながるわけではありません。フェイクアウト（騙し）は一般的です。
- ⚠️ 市場全体の方向性（SPY、QQQ）を確認してください。
- ⚠️ 決算日が近い場合は注意してください。ボラティリティが上昇します。
