# 黄金比
*すべての指標においてバランスの取れた「ゴルディロックス」銘柄を見つける*

## 1. この戦略とは何ですか？
- **リスクレベル**: 低〜中
- **Timeframe**: 1045
- **Target**: バランス型投資家、ポートフォリオ構築者

Golden Ratioは、どの指標においても極端ではない銘柄を見つけ出します：妥当なP/E（10-20）、中規模の時価総額（50億〜500億ドル）、適度な上昇（0.5-5%）、ピークまでの妥当な距離（60-85%）、および十分な出来高。このゴルディロックス戦略はバランスの取れたポイントであり、極端なリスクはありませんが、ポテンシャルを秘めています。

## 2. 自動フィルターパラメータ
このカードをクリックすると、以下のフィルターが自動的に適用されます：

| カラム | 適用値 |
| :--- | :--- |
| ライブ出来高 | > 1.000.000 |
| 時価総額 | > 10B |

## 3. 仕組み
P/E 10-20、時価総額 5,000-50,000M、変化率 0.5-5%、高値乖離率 60-85%、出来高 500,000以上。結果は変化率の降順で表示されます。

## 4. マニュアルチューニングガイド
スタイルに合わせてフィルターを手動で変更できます：
- P/E範囲を12〜18に設定すると、評価帯域を絞り込めます。
- 近接度（Proximity）の範囲を70〜80%に設定すると、より特定の領域に焦点を当てられます。
- 時価総額（Market Cap）の範囲を10,000〜30,000Mに設定すると、「スイートスポット」の範囲を絞り込めます。
- 変化率（Change）の制限を3%に下げると、極端なボラティリティを排除できます。

## 5. 実例シナリオ
免責事項：以下のシナリオは、説明のみを目的とした純粋に仮定の架空のデータを使用しています。実際の市場状況、実際の価格変動、または真の過去の記録を反映するものではありません。仮定のシナリオ：中型公益事業会社（時価総額約250億ドル、P/E 16）が、適度な日次上昇、ピークへの適度な近接、妥当な出来高といったバランスの取れた指標を示した場合、Golden Ratioはそれを「ゴルディロックス（適温）」なプロファイル、つまり高すぎず、安すぎず、ボラティリティも高すぎない銘柄としてフラグを立てます。

## 6. リスクと警告
- ⚠️ 指標が均衡しているということはリスクが低いことを意味しますが、大きなリターンを得るには時間が必要です。
- ⚠️ この戦略は、積極的なリターンを求める方には遅すぎる可能性があります。
- ⚠️ 市場全体が下落した場合、バランスの取れた銘柄も影響を受けます。相関性が高いためです。
