# 蓄積ゾーン
*価格は横ばいですが、出来高が急増しています。大口投資家の動きを追跡しましょう。*

## 1. この戦略とは何ですか？
- **リスクレベル**: 中
- **Timeframe**: (5-20)
- **Target**: テクニカルアナリスト、スマートマネー・フォロワー

Accumulation Zone（蓄積ゾーン）は、価格がほとんど変化していない（-0.5%から+0.5%）にもかかわらず、出来高が300万を超えている銘柄を特定します。これは「スマートマネーの蓄積」の兆候です。機関投資家が価格を動かすことなく、静かにポジションを構築している可能性があります。

## 2. 自動フィルターパラメータ
このカードをクリックすると、以下のフィルターが自動的に適用されます：

| カラム | 適用値 |
| :--- | :--- |
| ライブ出来高 | > 1.000.000 |
| 時価総額 | > 10B |

## 3. 仕組み
変化率が-0.5%から+0.5%の間、出来高が最低3,000,000、かつ時価総額が最低5,000M。結果は出来高の降順で表示されます。

## 4. マニュアルチューニングガイド
スタイルに合わせてフィルターを手動で変更できます：
- 範囲を-0.2%から+0.2%に変更すると、超安定的な蓄積ゾーンを表示できます。
- 出来高の最小値を5Mに引き上げると、メガボリュームのゾーンを見つけることができます。
- 高値からの距離の最小値を70%に追加して、強力な銘柄における蓄積を表示しましょう。
- 時価総額の最小値を20,000Mに引き上げると、大企業における蓄積に注目できます。

## 5. 実例シナリオ
免責事項：以下のシナリオは、説明のみを目的とした純粋に仮定の架空のデータを使用しています。実際の市場状況、実際の価格変動、または真の過去の記録を反映するものではありません。仮定のシナリオ：あるメガキャップのハイテク株が特定の日にわずか0.1％しか上昇しなかったものの、通常の2倍の出来高で取引された場合、出来高を伴う横ばいの価格は、機関投資家による静かな蓄積（アキュムレーション）を示唆している可能性があります。これは、Accumulation Zoneが検出するように設計された「目に見えない」シグナルです。

## 6. リスクと警告
- ⚠️ 高出来高かつ低価格変動は、ディストリビューション（売り抜け）の可能性もあり、方向性は不透明です。
- ⚠️ 蓄積シグナル後の動きには数日から数週間かかる場合があり、忍耐が必要です。
- ⚠️ 出来高は、オプションの満期日などの技術的な理由で膨らむこともあります。
