# ヒドゥン・ジャイアント
*誰もが見落としている中型株の成長候補を発掘*

## 1. この戦略とは何ですか？
- **リスクレベル**: 中
- **Timeframe**: (530)
- **Target**: グロースハンター、アクティブ投資家

ヒドゥン・ジャイアント戦略は、時価総額20億ドルから200億ドルの間で、低いP/E比率で評価され、52週高値の75%以上を維持している強い銘柄を見つけ出します。これらの企業はメガキャップほど目立ちませんが、強力なファンダメンタルズを備えており、市場のレーダー外にある成長マシンです。

## 2. 自動フィルターパラメータ
このカードをクリックすると、以下のフィルターが自動的に適用されます：

| カラム | 適用値 |
| :--- | :--- |
| ライブ出来高 | > 1.000.000 |
| 時価総額 | > 10B |

## 3. 仕組み
時価総額が2,000M〜20,000Mの間、P/Eレシオが0.1〜15、高値からの近接度が最低75%、かつ変化率が最低+0.1%。結果はP/Eレシオの昇順で表示されます。

## 4. マニュアルチューニングガイド
スタイルに合わせてフィルターを手動で変更できます：
- 時価総額の範囲を5,000〜15,000Mに設定すると、より厳選された中型株のスキャンが可能です。
- PERの下限を10に下げて、超割安な隠れた巨人を探します。
- 近接度の最小値を85%に引き上げると、ブレイクアウトに最も近い候補が見つかります。
- セクターフィルターを変更して、特定のセクター内の隠れた巨人を探してください。

## 5. 実例シナリオ
免責事項：以下のシナリオは、説明のみを目的とした純粋に仮定の捏造されたデータを使用しています。これは実際の市場状況、実際の価格変動、または真の過去の記録を反映するものではありません。仮定のシナリオ：中型株の消費者ブランド企業（時価総額150億ドル）のP/Eが12で、過去最高の収益を上げながらピークの82%付近にある場合、Hidden Giantは、メガキャップ株の影に隠れて見落とされがちな、注目すべき機会としてそれを強調します。

## 6. リスクと警告
- ⚠️ 中型株はメガキャップよりもボラティリティが高い傾向があります。
- ⚠️ 低いP/Eと中型株の組み合わせは、成長の鈍化を示唆している可能性があります。収益の傾向を確認してください。
- ⚠️ メガキャップよりも流動性が低い場合があります。大きなポジションを持つ際はスプレッドに注意してください。
