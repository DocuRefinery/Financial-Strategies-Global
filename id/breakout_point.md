# Titik Breakout
*Deteksi formasi breakout klasik secara langsung*

## 1. Apa Strategi Ini?
- **Tingkat Risiko**: Tinggi
- **Timeframe**: Jangka Pendek (1-5 hari)
- **Target**: Analis Teknis

Breakout Point menggabungkan tiga filter kritis: kedekatan 90%+ ke puncak, volume 1M+, dan perubahan harian positif. Kombinasi ini adalah suar sinyal untuk "breakout" klasik dalam analisis teknikal.

## 2. Parameter Auto-Filter
Saat Anda mengklik kartu ini, filter berikut diterapkan otomatis:

| Kolom | Nilai Diterapkan |
| :--- | :--- |
| Volume Langsung | > 1.000.000 |
| Kap. Pasar | > 10B |

## 3. Bagaimana Cara Kerjanya?
Proximity to Peak min 90%, Volume min 1.000.000, dan Change min +0,1%. Hasil dicantumkan menurun berdasarkan kedekatan.

## 4. Panduan Penyesuaian Manual
Anda dapat mengubah filter secara manual sesuai gaya Anda:
- Tingkatkan kedekatan ke 95% untuk melihat kandidat breakout yang paling matang.
- Tingkatkan volume minimum ke 2 juta untuk memperkuat konfirmasi institusional.
- Tingkatkan perubahan minimum ke 1% untuk melihat siapa yang mengalami breakout dengan momentum kuat.
- Tambahkan kapitalisasi pasar minimum 5.000 juta untuk fokus pada breakout di perusahaan yang lebih besar.

## 5. Skenario Dunia Nyata
SANGGAHAN: Skenario berikut menggunakan data hipotetis dan buatan semata-mata untuk tujuan ilustrasi. Ini TIDAK mencerminkan kondisi pasar yang sebenarnya, pergerakan harga aktual, atau catatan historis yang nyata. Skenario hipotetis: Jika saham e-commerce mega-cap berada 93% di dekat puncaknya, naik 1,8% secara intraday dengan volume 8 juta, ketiga kriteria Breakout Point akan terpenuhi—menandai formasi breakout klasik untuk tinjauan teknis lebih lanjut.

## 6. Risiko & Peringatan
- ⚠️ Tidak setiap sinyal breakout berubah menjadi breakout sungguhan—fakeout sering terjadi.
- ⚠️ Periksa arah pasar secara umum (SPY, QQQ).
- ⚠️ Berhati-hatilah jika tanggal pengumuman laba sudah dekat—volatilitas meningkat.
