# Ausbruchspunkt
*Klassische Breakout-Formation live erkennen*

## 1. Was ist diese Strategie?
- **Risikoniveau**: Hoch
- **Timeframe**: Kurzfristig (1-5 Tage)
- **Target**: Technischer Analyst, Breakout-Trader

Der Ausbruchspunkt kombiniert drei Filter: 90%+ Nähe zum Hoch, 1M+ Volumen und positive Tagesveränderung. Diese Dreifachkombination ist das Signalzeichen für eine klassische "Breakout"-Formation.

## 2. Auto-Filter-Parameter
Wenn Sie auf diese Karte klicken, werden die folgenden Filter automatisch angewendet:

| Säule | Angewandter Wert |
| :--- | :--- |
| Live-Volumen | > 1.000.000 |
| Marktkapitalisierung | > 10B |

## 3. Wie es funktioniert
Nähe zum Hoch min. 90%, Volumen min. 1.000.000, Veränderung min. +0,1%. Absteigend nach Nähe sortiert.

## 4. Leitfaden zur manuellen Abstimmung
Sie können die Filter manuell an Ihren Stil anpassen:
- Erhöhen Sie die Nähe auf 95% für die reifsten Breakout-Kandidaten.
- Erhöhen Sie Volumen auf 2M zur Stärkung der institutionellen Bestätigung.
- Erhöhen Sie Veränderung auf 1% für starkes Momentum beim Ausbruch.
- Fügen Sie Marktkap. min. 5.000M hinzu für Ausbrüche bei größeren Unternehmen.

## 5. Szenario aus der realen Welt
AMZN ist 93% nahe dem Hoch, stieg heute um 1,8% bei 8M Volumen. Alle drei Kriterien erfüllt — klassische Breakout-Formation. Einstiegspunkt.

## 6. Risiken & Warnungen
- ⚠️ Nicht jedes Breakout-Signal wird ein echter Ausbruch — Fehlausbrüche sind häufig.
- ⚠️ Prüfen Sie die allgemeine Marktrichtung (SPY, QQQ).
- ⚠️ Vorsicht bei nahendem Earnings-Termin — Volatilität steigt.
