# Wertschatz
*Vom Markt übersehene günstige Riesen finden*

## 1. Was ist diese Strategie?
- **Risikoniveau**: Niedrig
- **Timeframe**: Langfristig (1-12 Monate)
- **Target**: Value-Investor, Langfristanleger

Wertschatz findet Unternehmen mit großer Marktkapitalisierung und niedrigem KGV. Die digitale Version des Warren-Buffett-Value-Investing. Entdecken Sie fundamental starke Riesen, die der Markt übersieht.

## 2. Auto-Filter-Parameter
Wenn Sie auf diese Karte klicken, werden die folgenden Filter automatisch angewendet:

| Säule | Angewandter Wert |
| :--- | :--- |
| Live-Volumen | > 1.000.000 |
| Marktkapitalisierung | > 10B |

## 3. Wie es funktioniert
KGV maximal 12, Marktkapitalisierung mindestens 10.000M (10 Mrd. $). Ergebnisse aufsteigend nach KGV sortiert.

## 4. Leitfaden zur manuellen Abstimmung
Sie können die Filter manuell an Ihren Stil anpassen:
- Senken Sie das KGV-Limit auf 8, um ultra-günstige Riesen zu sehen.
- Erhöhen Sie die Mindest-Marktkapitalisierung auf 50.000M für nur Mega-Cap-Unternehmen.
- Setzen Sie den Sektorfilter auf "Finance" für Value-Chancen im Bankensektor.
- Setzen Sie Veränderung auf -2 bis +2, um ruhige, stabile Riesen zu sehen.

## 5. Szenario aus der realen Welt
BRK.B (Berkshire Hathaway) hat ein KGV von 9,5 und eine Marktkapitalisierung von 750 Mrd. $. Während sich der Markt auf die Tech-Rallye konzentriert, handelt dieser Riese mit Abschlag.

## 6. Risiken & Warnungen
- ⚠️ Niedriges KGV bedeutet nicht immer "günstig" — vergleichen Sie mit dem Sektordurchschnitt.
- ⚠️ Manche Unternehmen haben aus gutem Grund ein niedriges KGV (Stagnation, Schulden usw.).
- ⚠️ Vernachlässigen Sie die Bilanzanalyse nicht, um "Value Traps" zu vermeiden.
