# Bodensprung
*Aktien mit V-Umkehr und Erholungssignal einfangen*

## 1. Was ist diese Strategie?
- **Risikoniveau**: Hoch
- **Timeframe**: Mittelfristig (5-30 Tage)
- **Target**: Value-Investor, Contrarian-Trader

Die Bodensprung-Strategie identifiziert Aktien, die mehr als 50% von ihrem 52-Wochen-Hoch gefallen sind, aber heute positiv schlossen. Dies ist eine klassische V-Umkehr-Formation — die Erholung vom Tiefpunkt könnte begonnen haben.

## 2. Auto-Filter-Parameter
Wenn Sie auf diese Karte klicken, werden die folgenden Filter automatisch angewendet:

| Säule | Angewandter Wert |
| :--- | :--- |
| Live-Volumen | > 1.000.000 |
| Marktkapitalisierung | > 10B |

## 3. Wie es funktioniert
Nähe zum Hoch maximal 50% (weit vom Hoch), Veränderung (%) mindestens +0,1 (heute grün), Volumen mindestens 500.000.

## 4. Leitfaden zur manuellen Abstimmung
Sie können die Filter manuell an Ihren Stil anpassen:
- Senken Sie das Nähe-Maximum auf 30%, um tiefere Bodenumkehrungen zu finden.
- Erhöhen Sie das Veränderungs-Minimum auf 3%, um nur starke Erholungen zu sehen.
- Setzen Sie Marktkapitalisierung auf mindestens 5.000M für geringeres Insolvenzrisiko.
- Fügen Sie einen KGV-Filter hinzu, um profitable Unternehmen zu separieren.

## 5. Szenario aus der realen Welt
META ist 45% vom 52-Wochen-Hoch gefallen, stieg aber heute um 4%. Neuer Nutzerwachstumsbericht veröffentlicht. Das Umkehrsignal ist stark — frühe Einstiegsgelegenheit.

## 6. Risiken & Warnungen
- ⚠️ Was wie eine Bodenumkehr aussieht, könnte ein "Dead Cat Bounce" sein.
- ⚠️ Verlassen Sie sich nicht nur auf technische Signale ohne Fundamentalanalyse.
- ⚠️ Viele dieser Aktien sind aus gutem Grund gefallen — prüfen Sie die Finanzlage.
