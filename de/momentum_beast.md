# Momentum-Monster
*Der Trend ist dein Freund — starke Anstiege einfangen*

## 1. Was ist diese Strategie?
- **Risikoniveau**: Hoch
- **Timeframe**: Kurzfristig (1-5 Tage)
- **Target**: Momentum-Trader, Swing-Trader

Momentum-Monster findet Aktien mit über 5% Tagesanstieg und über 1 Million Volumen. Momentum-Strategien basieren auf der Annahme, dass ein starker Trend anhalten wird.

## 2. Auto-Filter-Parameter
Wenn Sie auf diese Karte klicken, werden die folgenden Filter automatisch angewendet:

| Säule | Angewandter Wert |
| :--- | :--- |
| Live-Volumen | > 1.000.000 |
| Marktkapitalisierung | > 10B |

## 3. Wie es funktioniert
Veränderung mindestens +5%, Volumen mindestens 1.000.000. Ergebnisse absteigend nach Veränderung sortiert.

## 4. Leitfaden zur manuellen Abstimmung
Sie können die Filter manuell an Ihren Stil anpassen:
- Erhöhen Sie die Mindestveränderung auf 8% für nur "Rallye"-Level-Anstiege.
- Erhöhen Sie das Mindestvolumen auf 3M zur Bestätigung institutioneller Beteiligung.
- Fügen Sie Nähe zum Hoch mindestens 80% hinzu für Momentum-Aktien im Trend.
- Fügen Sie Marktkap. mindestens 5.000M hinzu für Momentum bei größeren Unternehmen.

## 5. Szenario aus der realen Welt
TSLA stieg heute um 8% bei 15M Volumen. Neue Fabrik-Ankündigung. Momentum-Monster fängt dieses Signal — der Trend könnte anhalten.

## 6. Risiken & Warnungen
- ⚠️ Momentum-Strategien sind anfällig für "Late Entry"-Risiko.
- ⚠️ Gewinnmitnahmen nach extremem Anstieg können scharf sein.
- ⚠️ Verwenden Sie einen Trailing-Stop-Loss zum Gewinnschutz.
- ⚠️ Verwechseln Sie nicht allgemeines Marktmomentum mit einer einzelnen Aktie.
